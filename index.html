Terraforming Planet — generator wizualizacji

Proste demo: GitHub Pages jako frontend, Cloudflare Worker jako backend z kluczem API. Najprostsza architektura: GitHub Pages (HTML/JS) + Worker (secrets) + OpenAI Image Gen API.

Architektura

  1. Android (przeglądarka)
  2. GitHub Pages (HTML + JS)
  3. Cloudflare Worker (API key)
  4. OpenAI Image Gen API

Podmień URL Workera w app.js.

Ustawienia obrazu

Klucz API jest przechowywany wyłącznie w Workerze. Repozytorium nie zawiera sekretów.

Wynik

Gotowy do generowania. Kliknij Generate.

Instrukcja uruchomienia

1. GitHub Pages

  • Wrzuć pliki: index.html, style.css, app.js.
  • Settings → Pages → Source: main, folder: /root.

2. Cloudflare Worker

  • Dodaj kod Workera (plik worker.js).
  • Ustaw sekret OPENAI_API_KEY.

3. Android

  • Otwórz adres GitHub Pages w Chrome/Firefox.
  • Kliknij Generate i zobacz wynik.
